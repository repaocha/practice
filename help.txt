导出数据三种方法


准备工作：
1，一个exfat格式的硬盘，尽量容量大一点（以要导出的数据大小为参考依据）
2，一台可以正常上网的Mac电脑（可选）
3，火线或者雷线（可官网购买）


方法：
1，开机按住command＋r，进入到实用工具界面，进入磁盘工具，把硬盘连上设备，在磁盘工具中选中系统盘，左上角有“文件”选项－“新建印像“－选择最后一个来自XX（磁盘名）的映像，路径选择硬盘。



2，在准备的硬盘内安装一个OS X系统，硬盘连上电脑，开机按住option，选择硬盘内的系统启动电脑，启动后在finder去把文件拖到硬盘内就可以。
从 App Store 下载 OS X 安装器


请执行以下步骤下载要安装的 OS X 版本。

1. 从 Apple 菜单中，选取“App Store”。

2. 按住 Option 键并点按“已购项目”标签。您应该可以看到您以前购买的 OS X 版本。如果您看不到您的已购项目，请使用您以前用于安装 OS X 的 ID 登录 App Store。（或是直接在App Store中搜索想要升级的版本）

3. 点按您要下载的 OS X 版本旁的“安装”按钮。

Mac App Store 通常会在“已购项目”面板中显示 OS X 为“已安装”。当您的 Mac 已经安装有相同版本的 OS X 时，这会阻止您再次下载。当您在点按“已购项目”标签的同时按住 Option 键时，您要下载的项目右侧的按钮应该会变成“安装”。如果此按钮未改变，则使用 Spotlight 在您的电脑上搜索“安装 OS X”。有可能在您的“应用程序”文件夹或连接的驱动器上已有相同的 OS X 安装器的副本。



3，（最快的一个方法，但是需要购买工具：火线或雷线，不推荐）两台电脑用线缆连接上，开机按住d，会出现y字图标与闪电图标，后面就会进入传输模式，选择文件就可以。
